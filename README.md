# E-commerce Customer Analytics

## Project Overview

This project analyzes customer purchasing behavior using the Online Retail II dataset. The analysis helps identify top customers, best-selling products, sales trends, and purchasing patterns that businesses can use to improve decision-making.

---

## Dataset

- Online Retail II Dataset
- Source: Kaggle
- Format: Excel (.xlsx)

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## Project Objectives

- Analyze customer purchasing behavior
- Identify top customers by revenue
- Discover best-selling products
- Analyze monthly sales trends
- Study sales by country
- Analyze customer purchasing hours
- Understand order value distribution

---

## Visualizations

- Top Customers by Revenue
- Monthly Sales Trend
- Sales by Country
- Orders by Hour
- Best Selling Products
- Top Products by Revenue
- Order Value Distribution

---

## Business Insights

- A small group of customers generates a significant portion of revenue.
- The United Kingdom contributes the highest sales.
- A few products account for most of the total revenue.
- Customer purchasing follows clear seasonal patterns.
- Most orders are small, while a few high-value orders contribute heavily to total sales.

---

## Business Recommendations

- Reward loyal customers through loyalty programs.
- Increase stock for best-selling products.
- Launch promotions during peak shopping hours.
- Focus marketing on high-performing countries.
- Encourage larger purchases through bundle offers.

## How to Run

1. Clone the repository.
2. Install the required libraries:
   ```
   pip install pandas numpy matplotlib openpyxl
   ```
3. Open `ecommerce_analysis.ipynb`.
4. Run all notebook cells.
